About the project

Map that users can create custom markers with custom names and descriptions. Markers can be moved by drag and drop, edited, and deleted. Map shows various other assets stored in the database and filter by tag or distance from the current location.

Date
October 1, 2021
My Role
Full responsibility for the front-end. Create a Map component and action/reducers for all the map features and work closely together with the backend developer.

Map with various features

  • Set or change locations of assets
  • Change map types - street, satellites, etc.
  • Set custom markers
  • Filter by types to show/hide certain types of assets
  • Filter by tags and distance to show/hide assets
